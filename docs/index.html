Projet Laravel portfolio

ImmoRadar aide à comparer un bien immobilier sans se laisser piéger par le coup de cœur.

Crée un projet, ajoute les biens manuellement, prépare la visite, estime le coût réel mensuel, lis les alertes et génère un rapport PDF d'aide à la décision.

Voir le dépôt GitHub Retour portfolio

GitHub Pages affiche cette page de présentation statique. L'application complète est une app Laravel/PHP avec authentification et base de données : elle doit être lancée en local ou déployée sur un hébergement PHP.

Ce que l'app rend lisible

Simple pour l'utilisateur

Projet, biens, checklist, coût réel, verdict. Pas d'import magique, pas de scraping, pas d'IA.
